## 第 02 章：基本查詢 SELECT {#第02章}

**2-1**
```sql
SELECT name, nationality
FROM authors;
```

**2-2**
```sql
SELECT
    title AS 書名,
    price AS 售價
FROM products;
```

**2-3**
```sql
SELECT
    title        AS 書名,
    price        AS 原價,
    price * 0.9  AS 優惠價
FROM products;
```

**2-4**
```sql
SELECT DISTINCT city
FROM customers;
```

**2-5**
```sql
SELECT
    name || '（' || nationality || '）' AS 作者資訊
FROM authors;
```

## 第 03 章：條件篩選 WHERE {#第03章}

**3-1**
```sql
SELECT title
FROM products
WHERE stock = 0;
-- 結果：死水微瀾
```

**3-2**
```sql
SELECT title, published_year
FROM products
WHERE published_year >= 2000;
```

**3-3**
```sql
SELECT title, price
FROM products
WHERE price BETWEEN 300 AND 400;
```

**3-4**
```sql
SELECT name, city
FROM customers
WHERE city IN ('台北', '新竹');
```

**3-5**
```sql
SELECT title
FROM products
WHERE title LIKE '%的%';
-- 結果：挪威的森林、海邊的卡夫卡、單車失竊記（無「的」）
-- 實際含「的」：挪威的森林、海邊的卡夫卡
```

**3-6**
```sql
SELECT name, nationality
FROM authors
WHERE nationality NOT IN ('日本', '中國');
-- 結果：吳明益（台灣）、J.K. 羅琳（英國）、安潔拉·達克沃斯（美國）、馬克·吐溫（美國）
```

**3-7**（挑戰）
```sql
SELECT title, price, stock
FROM products
WHERE stock > 0 AND price < 300;
-- 結果：活著（280, 60）、傳說（290, 15）
```

## 第 04 章：排序與限制 {#第04章}

**4-1**
```sql
SELECT title, published_year
FROM products
ORDER BY published_year DESC;
```

**4-2**
```sql
SELECT title, stock
FROM products
ORDER BY stock DESC
LIMIT 3;
```

**4-3**
```sql
SELECT title, price
FROM products
WHERE category_id = 1
ORDER BY price ASC;
```

**4-4**
```sql
-- 第 2 頁（每頁 4 本，跳過前 4 筆）
SELECT title
FROM products
ORDER BY product_id
LIMIT 4 OFFSET 4;
```

**4-5**
```sql
SELECT title, price, published_year
FROM products
WHERE published_year > 2010
ORDER BY price DESC
LIMIT 2;
```

## 第 05 章：聚合函數與分組 {#第05章}

**5-1**
```sql
SELECT
    COUNT(*)             AS 書籍總數,
    SUM(stock)           AS 總庫存,
    ROUND(AVG(price), 0) AS 平均售價
FROM products;
```

**5-2**
```sql
SELECT
    nationality AS 國籍,
    COUNT(*)    AS 作者數
FROM authors
GROUP BY nationality
ORDER BY 作者數 DESC;
```

**5-3**
```sql
SELECT
    status   AS 訂單狀態,
    COUNT(*) AS 訂單數量
FROM orders
GROUP BY status;
```

**5-4**
```sql
SELECT
    category_id,
    ROUND(AVG(price), 0) AS 平均售價
FROM products
GROUP BY category_id
HAVING AVG(price) > 300;
```

**5-5**
```sql
SELECT
    city     AS 城市,
    COUNT(*) AS 顧客人數
FROM customers
GROUP BY city
HAVING COUNT(*) >= 2
ORDER BY 顧客人數 DESC;
-- 結果：台北 3
```

**5-6**（挑戰）
```sql
SELECT
    category_id,
    COUNT(*)             AS 有庫存書籍數,
    ROUND(AVG(price), 0) AS 平均售價
FROM products
WHERE stock > 0
GROUP BY category_id;
```

## 綜合練習 {#綜合}

**S-1**
```sql
SELECT title, price, stock
FROM products
ORDER BY price ASC, title ASC;
```

**S-2**
```sql
SELECT
    COUNT(*)             AS 書籍數量,
    ROUND(AVG(price), 0) AS 平均售價
FROM products
WHERE published_year >= 2020;
```

**S-3**
```sql
SELECT title, price
FROM products
WHERE price > (SELECT AVG(price) FROM products)
ORDER BY price DESC;
```
